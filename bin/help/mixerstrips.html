Mixer Strips

Order

The order of the Mixer strips is not random, and it is not ordered by how the instruments were added either. The sorting order has three stages:
  1. Sort by connections. This ensures that the audio flow always goes from left to right.
  2. Sort by vertical position in the modular mixer.
  3. Sort by horizontal position in the modular mixer.

How to...

Q: Set accurate volum by writing a number?
A: Click the number box right above the volume slider

Q: Reset highest peak value in the box above the audio meter?
A: Just click the box.

Q: Add a new mixer strip?
A: You can not do that directly in a mixer strips window. You either have to create a new instrument in the modular mixer, or add a new instrument to a track in the editor.

Q: Delete a mixer strip?
A: Right-click the name of the mixer strip, and select "Delete".
