What is timing?

Timing is tempo and rythm.

What does Radium use timing for?

  1. External plugins (For instance to a make a VST plugin play echo sounds in sync with the music.)
  2. The metronome (Enabled by the "Click" button in the bottom bar)
  3. Modulators
  4. The sequencer grid 






How is timing controlled?


Timing is controlled by tempo events and signature events:

    

  •  Tempos (BPM) specify the duration of a quarter note. Note that BPM stands for Beats Per Minute, which is a misleading name unless your signature is 4/4. QPM (Quarters Per Minute) would be more correct.

  •  Signatures specify rythm. It does so by specifying number of beats divided by bar duration (for instance 4/4). Bar duration is measured in number of quarter notes.




For example, if tempo is set to 64 BPM, and the signature is set to 8/8, a new bar will be played every 16 seconds (64/4 * 8/8), and a new beat will be played every 2 seconds (64/4 * 1/8).
If the signature is set to 2/4, a new bar will be played every 8 seconds (64/4 * 2/4), and a new beat will be played every 4 seconds (64/4 * 1/4).








Timing modes


There are two timing modes to select from. Only one of them can be enabled at the same time:


    

  1. Editor timing
    

    
Editor timing is the default mode. In editor timing mode, tempos and signaturs are provided by the editor blocks placed in the first seqtrack.

    
Tempos are edited in the tempo tracks in the editor, and signatures are edited in the signature tracks in the editor.

    
In this mode, the tempo and signature lanes in the sequencer are just used for visualization, and editing in any of these two lanes is disabled.


    


  2. Sequencer timing
    

    
In sequencer timing mode, tempos and signatures are provided by the tempo and signature lanes in the sequencer.
This mode can be set explicitly in the popup menu of the sequencer,
but it will also be used automatically if the first seqtrack is for audio and not for editor blocks

    
If you are using Radium mainly for recording and playing back audio files,
this mode is both simpler, and probably easier to use, than editor timing.

    
Note that accelerando and ritardando is not currently supported in this mode.
Two more limitations of the sequencer timing mode is that signatures must align with bars, and that tempos must align with beats.
Editor timing mode does not have these three limitations.




To select timing mode, use the popup menu in the sequencer and either select "Editor timing" or "Sequencer timing".
Note that if the first seqtrack is for audio files, you are always forced to use sequencer timing.
