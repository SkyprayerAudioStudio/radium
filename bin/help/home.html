Main GUI

  1. Menus
    • Use the left alt key to show/hide menus and start/stop navigitating the menus with the keyboard.
    • You can also stop navigating the menus by pressing Esc.
    • Hold shift to quicker navigate up and down when using the up and down arrow keys.
  2. Editor
    • The editor consists of tempo tracks and note tracks. In the lower left area, there is a slider to change tempo in realtime, and a track scrollbar.
    • Press F12 to assign an instrument to the current track.
    • Press Right Shift to play current block.
    • Press right mouse button to erase notes in the piano roll.
    • Some documentation for the note tracks:
      • FX text
      • Chance text
      • Swing text
      • Velocity text
  3. Blocklist
    The blocklist shows a list of all available blocks.
    • Click a block to make it the current block.
    • Doubleclick a block to add it to the playlist.
    • Shift + rightclick a block to delete a block.
  4. Playlist
    The playlist is a textual representation of the current seqtrack in the sequencer. The playlist does not display more information than the sequencer, but it can be more convenient to use, especially if there is only one seqtrack.
  5. Mixer

    Mixer widget header

    • A/B settings buttons
    • "I" button: Whether to show the instrument widget inside the mixer widget.
    • "W" button: Whether to show the mixer widget in a separate window.
    • "M" button: Whether to show the modular mixer or mixer strips.

    Modular Mixer

    General

    • Move objects with right mouse button.
    • Double-click the name of an object to open GUI.
    • Delete objects or connections by pressing SHIFT and click right.
    • Select more than one object by holding CTRL when clicking. (Use Cmd key instead of CTRL if using Mac)
    • Move and connect object in one operation: Place one object on top of another object.
    • Zoom in and out by pressing CTRL and using the scroll wheel.

    Mixer objects

    • "M": Mute
    • "S": Solo
    • "B": Bypass

    Mixer Strips

    Order

    The order of the strips is not random or chronological. The sorting logic has four stages:
    1. Sort by whether the strip is a bus or not. Non-buses are placed before buses.
    2. Sort by vertical position in the modular mixer (top -> bottom).
    3. Sort by horizontal position in the modular mixer (left -> right).
    4. Rearrange (as little as possible) so that the audio flow always goes from left to right.

    How to...

    Q: Set accurate volume by writing a number?
    A: Click the number box right above the volume slider

    Q: Reset highest peak value in the box above the audio meter?
    A: Just click the box.

    Q: Add a new mixer strip?
    A: You can not do that directly in a mixer strips window. You either have to create a new instrument in the modular mixer, add a new seqtrack in the sequencer, or add a new instrument to a track in the editor.

    Q: Switch between a narrow and a wide strip?
    A: Left-click the name of the current mixer strip. (if the mixer strip isn't current, click first to make it current.)

    Q: Delete a mixer strip?
    A: Right-click the name of the mixer strip, and select "Delete". Or shift+right click the name.

  6. Mixer strip for current instrument
    • Shows a mixer strip for the current instrument.
    • The current instrument is also shown in the instrument widget.
    • Input MIDI signals are sent to the current instrument.
  7. Main window lower tabs
    • Sequencer
      • The sequencer lets you select when a block starts playing.
      • Add a new sequencer track ("seqtrack") by clicking one of the "+ ..." buttons in the lower left corner.
      • Seqblocks:
        • Move a seqblock with the mouse.
        • To move several seqblocks, either mark a rectangle, or press Ctrl and click on the seqblocks you want to move.
        • To quickly copy a seqblock, hold the Alt key and drag a seqblock. On macOS: Hold the Ctrl key.
        • Double-click a seqblock to open a GUI for it.
      • Use the popup menu to add effect automations, tempo automations, loop start and end positions, and other operations.
      • To quickly start playing at the position of the mouse cursor, scroll the mouse wheel up, and stop by scrolling down.
      • To record audio, click the "R" button in the seqtrack header of audio seqtracks, and start playing. To configure number of audio channels and where to get audio from, right-click the "R" button. You might also want to select beforehand when to start and stop recording, and you can do this by by right-clicking in the seqblock area and select the "Punch in/out" option under "Sequencer timeline".
      • For information about the different types of fade in and fade out shapes, look in the Ardour manual here.
    • Instrument widget
      • The instrument widget contains several elements to control the sound. See tool tips for more information.
      • The term "Instrument" is used for all types of sound objects, for instance plugins, MIDI event manipulators, and sound buses.
      • The instrument widget is either positioned in:
        • The lower tabs in the main window
        • The lower area of the mixer window
        Click the "I" button in the mixer to switch between these two positions.
      • By right-clicking a slider or checkbox, you can learn MIDI, assign an LFO, add automation, record into the editor, etc.
      • Some common instruments:
        • Sample Player
          The sample player plays audio files.
          • The Sample Player can load XI intruments; Soundfonts, and all types of sample formats supported by libsndfile.
          • WAV files are looped if they have loops defined in the "sampl" chunk, or they have "Loop Start" and "Loop End" cue id's.
          • SoundFonts (.sf2 files) often sound better when played with the FluidSynth instrument instead of the Sample Player. However, the Sample Player uses less memory, are faster to create, has sample-accurate note scheduling, supports pitch changes and polyphonic aftertouch (velocity can be changed while a note is playing), and has configurable options such as attack, decay, sustain, and release.
          • There is currently no graphical sample editor to edit samples, a feature which traditionally are provided by trackers. This feature should be provided later though. Until then, external sample editors must be used instead, if needed.
          • To record audio, either from outside the program or inside the program, click the red Record button in the instrument widget header. A new audio file will be created automatically when recording. Recording will start when the new instrument receives a note, and it will stop recording when that note stops playing.
        • FaustDev
          The Faust instrument lets you program Faust instruments in Radium. Also note that there is no "compile" button since programs are compiled in realtime while you write code.
        • Pure Data
          The Pure Data instrument lets you program Pure Data instruments in Radium. Currently, this instrument is only available on Linux.
    • Edit
      The edit tab provides various functions to edit notes in the editor. Right click a button to edit the keybinding for it.
  8. Bottom bar
    The bottom bar contains elements that might be useful at all times.
    • The bottom bar is visible in:
      1. The bottom of the main window
      2. The bottom of the mixer, when the mixer runs in its own window.
    • Use tool tips to get more information about each element in the bottom bar.

Mouse

  • Finetuning:
    • Linux and Windows: Press the Ctrl key.
    • macOS: Press the Cmd key.
  • Y direction only:
    • All systems: Press Shift key.
  • X direction only:
    • Linux and Windows: Press on of the Windows (Meta) keys.
    • macOS: Press the Left Alt key.
  • Move all automation nodes:
    • Linux and Windows: Press the Alt key.
    • macOS: Press one of the Ctrl keys.
  • Reset automation:
    • Linux and Windows: Press Shift and Alt, and click Right mouse button.
    • macOS: Press Shift and Ctrl, and click Right mouse button.
  • Break grid:
    • Linux and Windows: Press the Ctrl key. (i.e. do finetuning)
    • macOS: Press the Cmd key. (i.e. do finetuning)
  • Delete object under mouse:
    • All systems: Press Shift and click Right mouse button.
      This works for automation node, sound object, plugin, track, block, note, etc.
  • Select/unselect something:
    • Linux and Windows: Press the Ctrl key and click Left mouse button.
    • macOS: Press the Cmd key and click Left mouse button.
  • Zoom in or out:
    • Linux and Windows: Press the Ctrl key and use the Mouse scroll wheel.
    • macOS: Press the Cmd key and use the Mouse scroll wheel.
  • Scroll vertically:
    • All systems: Use the Mouse scroll wheel.
    • Below the timeline area in the sequencer you also need to hold one of the Windows (Meta) keys (OSX: Left Alt key).
    • (Note that the editor works differently. In the editor, the mouse scroll wheel does the same as Arrow Down and Arrow Up, and those keys will zoom lines in/out, or move the cursor 8 steps up/down, depending on which Shift key is pressed. Note that you can configure the keybindings.)
  • Scroll horizontally:
    • Linux and Windows: Use the Mouse scroll wheel and press on of the Windows (Meta) keys.
    • macOS: Use the Mouse scroll wheel and press the Left Alt key.
    • (Note that the editor works differently. In the editor, the mouse scroll wheel does the same as Arrow Down and Arrow Up. By default the Arrow Down and Arrow Up keys are not assigned to any keybinding while holding a Windows key, while on macOS, the alt key is assigned to scroll to the previous or next event on the current track. Note that you can configure the keybindings.)
  • Play song
    • All systems: Use the Mouse scroll wheel inside the sequencer. Scroll up to play, scroll down to stop.

Terms

  • Block - What is known as a "pattern" in trackers. Another term for this is Editor Block.
  • Seqtrack - A sequencer track.
  • Seqblock - The blocks in seqtrack.
    • For editor seqtracks: A seqblock contains an editor block.
    • For audio seqtracks: A seqblock contains an audio files.
  • Velocity - The volume of a single note.
