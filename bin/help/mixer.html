Mixer

  • Move objects with right mouse button.
  • Double-click the name of an object to open GUI.
  • Delete objects or connections by pressing SHIFT and click left (or right).
  • Select more than one object by holding CTRL when clicking.
  • To autoconnect a new object to an existing object, right-click at
  • To move and connect an object in one operation, place one object on top of another object.
  • Zoom in and out by pressing CTRL and using the scroll wheel.
