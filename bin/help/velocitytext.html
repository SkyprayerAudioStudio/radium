Velocity text

Velocity text is an alternative way to edit the velocities of a note. Editing text is often quicker than editing velocities graphically.

Key bindings:

  • Left Alt + Y: Turn on/off velocity text for track
  • Left Ctrl + Y: Turn on/off velocity text for all tracks in block

Velocity text format: xxt, where 

xx = Velocity value. 0 = lowest velocity, ff = highest velocity. (hex format)
t = Whether to glide to the next effect or not (denoted by '|').

Tips:

  • To quickly add an ff node, press 'G'.
  • To switch glide mode (i.e. add or remove the | character), press 'T'.
