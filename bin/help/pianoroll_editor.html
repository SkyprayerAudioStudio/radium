Pianoroll

How to

  • Create note:
    1. Move cursor where you want the note to start.
    2. Press left mouse button.
    3. Move cursor.
    4. Release left mouse button.
  • Delete current note:
    1. Move cursor above the note you want to delete.
    2. Hold shift and press right mouse button.
  • Delete all selected notes:
    1. Move cursor above a selected note.
    2. Select "Delete all selected notes".
  • Move note:
    1. Move cursor above the note you want to move.
    2. Press left mouse button.
    3. Move cursor.
    4. Release left mouse button.

Selecting and deselecting notes

  • To select or deselect a single note: Press Ctrl + left mouse button
  • To deselect all notes: Click left mouse button anywhere inside the piano roll.
  • To create a selection rectangle: Press Ctrl and mark a rectangle with left mouse button

Clipboard

  • Copy: Press Ctrl + C
  • Cut: Press Ctrl + X
  • Paste: Press Ctrl + V

Rubber

The Rubber deletes any playable part of a note under the mouse cursor. Press right mouse button and drag to show the rubber.
