Pianoroll

How to

  • Create note:
    1. Move cursor where you want the note to start.
    2. Press left mouse button.
    3. Move cursor.
    4. Release left mouse button.
  • Delete current note:
    1. Move cursor above the note you want to delete.
    2. Hold shift and press right mouse button.
  • Delete all selected notes:
    1. Move cursor above a selected note.
    2. Select "Delete all selected notes".
  • Move note:
    1. Move cursor above the note you want to move.
    2. Press left mouse button.
    3. Move cursor.
    4. Release left mouse button.

Selecting and deselecting notes

  • To select or deselect a single note: Press Ctrl + left mouse button
  • To deselect all notes: Click left mouse button anywhere inside the piano roll.
  • To create a selection rectangle: Press Ctrl and mark a rectangle with left mouse button

Clipboard

  • Copy: Press Ctrl + C
  • Cut: Press Ctrl + X
  • Paste: Press Ctrl + V
Note that if the mouse cursor is above a single note while copying or cutting, and this note is not selected, this note will be copied/cut instead of the selected notes.

Rubber

The Rubber deletes any playable part of a note that is under the mouse cursor. Press right mouse button and drag to show the rubber.

Selection vs. range

There are two ways to select notes:
  1. Selection in the pianoroll.
  2. Ranged notes in the note-text subtrack: Define a note-range by pressing Left Extra + B.
  • These two ways don't share the same clipboard buffer.
  • A selection operates on a track, while a range operates on a block.
  • If there is no range in the current block, all functions operating on a range will instead work on the selected notes in the current track.
