FX text

FX text is an alternative way to edit effects (automation) in the editor. Editing text is often quicker than editing effects graphically.

You enable FX text by right clicking a track header and select "FX text".

FX text format: xxt, where 

xx = effect value. 0 = lowest value, ff = highest value. (hex format)
 t = Whether to glide to the next effect or not (denoted by '|').

A special situation:

Let's say you have an fx with two nodes. The first node has the value '6e|', and the second node has the value '28 '.
Q: Will the value 28 be sent to the instrument?
A: Yes.

Tips:

  • Left Alt + C/X/V Copies/Cuts/Pastes the current fx track.
  • To quickly add an ff node, press 'G'.
  • To switch glide mode (i.e. add or remove the | character), press 'T'.
